Voglio che tu agisca come un Senior Full Stack Developer specializzato in HTML, CSS e JavaScript Vanilla.

Devi progettare e generare il pannello ADMIN completo di un sito web per un ristorante a tema **Satisfactory** e **FICSIT**.

L'interfaccia deve richiamare lo stile industriale futuristico della FICSIT:

* Colori principali:

  * Arancione #FF7A00
  * Nero #111111
  * Grigio scuro #1C1C1C
  * Grigio chiaro #2F2F2F
  * Bianco #FFFFFF

* Design:

  * Futuristico
  * Industriale
  * Dashboard moderna
  * Effetti glow arancioni
  * Animazioni fluide
  * Responsive desktop/tablet/mobile
  * Font stile tecnologico

# Architettura del progetto

Genera tutti i file separatamente.

## principale.html

Funzione:

* Dashboard principale ADMIN
* Sidebar laterale sinistra
* Header superiore
* Area contenuti centrale

La dashboard deve caricare tutte le altre pagine tramite iframe senza ricaricare la pagina.

Menu laterale:

* Dashboard
* Gestione Menù
* Prenotazioni
* Ordinazioni
* Recensioni
* Logout

L'iframe deve occupare tutto lo spazio disponibile.

# Requisiti tecnici

Genera codice completo e funzionante.

NON utilizzare:

* Bootstrap
* Tailwind
* React
* Vue
* Angular
* jQuery

Utilizzare esclusivamente:

* HTML5
* CSS3
* JavaScript Vanilla ES6

Il codice deve essere:

* Commentato
* Pulito
* Scalabile
* Professionale
* Responsive

Genera ogni file in sezioni separate:

```txt
=== principale.html ===
codice

=== principale.css ===
codice

=== principale.js ===
codice
```

e così via per tutti i file.

Fornisci il contenuto completo di ogni file senza omettere nessuna parte del codice.
